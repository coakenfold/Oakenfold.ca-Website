Web Performance

Different strategies are required for the type of site that is being delivered. Strategic bundling of .js is a good idea for a single page application but it is not as relevant on a single page hand coded html/css site.

Successful strategies I have used in production:

  • require.js dynamic includes
  • lazy loading 3rd party
  • http/2
  • webpagetest
  • Cache static content, reduce expensive db calls, inline cache .json to rehydrate state
